# Data Preprocessing

This directory contains the preprocessing scripts for the datasets used in the AnyPPG project.

## Dataset Processing

Specific data processing instructions and scripts can be found in the corresponding subdirectories:

- `cfs/`
- `hsp/`
- `mc_med/`
- `mesa/`
- `pulsedb/`

## Data Availability

All raw datasets used in this project are publicly available. They can be obtained either by direct download or by submitting an application to the respective data providers.
